# 🌟 Nishant Nahar — Personal Portfolio

Welcome to my personal portfolio repository! This is a clean, modern, and responsive single-page portfolio website built to showcase my projects, skills, and experience as a developer. Designed with a premium aesthetic, it features a sleek black-and-white theme with striking gold accents.

> **Live Demo:** [Check it out here](https://its-nishant-10.github.io/portfolio_website/)

---

## ✨ Highlights

- **Hero Section**: Features a dynamic typewriter animation and a custom-drawn robot mascot SVG to create an engaging first impression.
- **Projects Showcase**: Interactive project cards that open into full-screen modals to provide detailed insights into my work.
- **Categorized Portfolio**: Easily filter projects by Web, Automation, and FOSS/Design.
- **About Me**: A personal biography section featuring fun animated sticky notes.
- **Skills & Tools**: Animated skill progress bars that trigger smoothly upon scrolling.
- **Responsive Design**: Carefully optimized layouts for seamless viewing across desktop, tablet, and mobile devices.
- **Custom Animations**: Polished UX with custom scroll behaviors and subtle fade-in animations as you navigate the page.

---

## 🛠️ Technology Stack

This portfolio is built with pure, performant web technologies—no bloated frameworks or heavy libraries:

- **HTML5**: Semantic and accessible structuring
- **CSS3**: Advanced styling using Flexbox, CSS Grid, Custom Variables, and Animations
- **Vanilla JavaScript**: Interactivity, DOM manipulation, IntersectionObserver API, and custom smooth scrolling
- **Font Awesome 6**: Vector icons
- **Google Fonts**: Space Grotesk, DM Sans, and IBM Plex Mono typography
